# mkapp

Convert executable to Mac OS X application bundle.

## Usage

```sh
usage: mkapp <executable> <image>

args:
    exec:  execuatble                ex: Icarus.sh
    image: 1024 x 1024 PNG image     ex: Icarus.png
```

## Example

```sh
./mkapp examples/Icarus.sh examples/Icarus.png
```
